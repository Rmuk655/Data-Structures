# Data Structures Project

Various Labassignments like 
    PostFixEvaluator
    ReverseQeue
    QueueWithTwoStacks
    PerfectHashFunction
    BinaryTree
    AVL Tree
    RedBlackTree
    HeapSort
    TwoCNFSAT (using graph)
## Getting Started
- Open this folder in Visual Studio Code.
- Ensure you have the gc++ and make installed.
- make targets are present for perfect_hash binary_tree avl_tree rb_tree twofour_tree twoCNFSAT
- build .c files directly using gcc
